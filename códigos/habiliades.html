Principais linguagens em que domino

Linguagem de programação de propósito geral, de baixo nível, conhecida pela eficiência e controle sobre o hardware.

Linguagem de programação da Microsoft, orientada a objetos, usada para desenvolver aplicativos para desktop, web e mobile.

Linguagem de estilo para controlar a apresentação de documentos HTML, definindo layout, cores e fontes em páginas da web.

Linguagem de marcação para criar a estrutura básica de páginas da web, definindo elementos e organizando conteúdo.

Linguagem de programação da web, usada para adicionar interatividade a páginas da web, executada no navegador do usuário.

Sistema de gerenciamento de banco de dados relacional, de código aberto, usado para armazenar e gerenciar dados em aplicações web.

Linguagem de script de servidor, usada para desenvolver aplicativos web dinâmicos e interativos, integrando-se com HTML.

Linguagem de consulta estruturada, usada para gerenciar bancos de dados relacionais, executando operações como inserção, atualização e consulta de dados.
